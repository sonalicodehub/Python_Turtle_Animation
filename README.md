# 🐢 Python Turtle Graphics

A creative Python project built using the **Turtle Graphics** library to generate colorful geometric artwork through animation. This project demonstrates the power of Python's built-in graphics module by combining loops, rotational symmetry, and color cycling to create visually appealing designs.

## ✨ Features

* 🐢 Built with Python Turtle Graphics
* 🎨 Generates colorful geometric patterns
* 🌈 Dynamic color cycling
* 🔄 360° rotational animation
* ⚡ Smooth drawing using animation optimization
* 📐 Demonstrates loops and geometric transformations
* 🎓 Beginner-friendly Python graphics project

## 🎮 How It Works

* A turtle object draws a circle with a fixed radius.
* After each circle, the turtle rotates by **1°**.
* The pen color changes automatically using a predefined color palette.
* The process repeats **360 times**, producing a beautiful circular design.
* The final output showcases geometric symmetry and creative coding.

## 🛠️ Tech Stack

* Python 3.x
* Turtle Graphics (Built-in Python Library)
